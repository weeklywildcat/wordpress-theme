# Theme Modules

Weekly Wildcat ships reusable native block patterns for common newspaper modules.

## Homepage modules

- Breaking news banner
- Lead story and editor picks
- Section front module
- Photo feature strip
- Newsletter or announcement callout

## Editing workflow

Use **Appearance → Editor → Patterns** to insert or edit modules. These modules are intentionally block patterns rather than custom PHP settings so editors can rearrange them with native WordPress tools.

## Accessibility notes

- Keep button and link text descriptive.
- Do not rely on color alone for urgent labels.
- Keep headings in logical order when moving modules.
- Add alt text to featured images in the Media Library.
